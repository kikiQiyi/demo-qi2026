# Thai Sacred Sites and Religious Objects

This project is a small digital collection built with CollectionBuilder-CSV. It documents sacred sites, religious objects, offering practices, and devotional spaces in Thailand, with a focus on Bangkok, Chiang Rai, and Mae Sai.

The collection includes photographs of Buddhist temples, Chinese-Thai shrines, amulet markets, donation boxes, offering sets, Buddha images, and religious objects sold in market spaces. Together, these items show how religion appears not only in formal temple settings, but also in everyday spaces of commerce, travel, and public devotion.

## Project Theme

The main theme of this collection is the relationship between sacred spaces and material religion in Thailand. Rather than focusing only on temple architecture, the project also looks at objects and practices connected to religious life, including:

- Buddhist amulets
- Buddha statues
- donation boxes
- offering baskets
- incense and candles
- shrine altars
- guardian figures
- religious market displays

These objects help show how religious practice is experienced visually, materially, and socially.
